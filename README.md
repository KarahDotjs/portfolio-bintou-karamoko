## 📁 Structure du projet

```
src/
├── components/
│   ├── CircuitBackground.vue  # SVG décoratif animé
│   ├── SiteNavbar.vue         # Navigation responsive
│   ├── PortfolioMain.vue      # Sections principales
│   └── SiteFooter.vue         # Footer
├── App.vue                    # Composant racine
└── main.js                    # Point d'entrée
```

---

## 🚀 Lancer le projet en local

```bash
# Cloner le repo
git clone https://github.com/KarahDotjs/portfolio-bintou-karamoko.git

# Installer les dépendances
npm install

# Lancer le serveur de dev
npm run dev

# Build de production
npm run build
```
